# Fraud Detection Project

Hello! This is my machine learning project to predict and detect fraud. The main goal of this notebook is to look at transaction data and guess which transactions are normal and which are fraud. 

## What I do in this project
Fraud detection is tricky because there are usually very few fraud cases compared to normal cases. This means the data is highly imbalanced. 

In this project, I clean the data and prepare it for machine learning. Then, I train models to catch the fraud.

The machine learning models I try in this project are:
* [Model 1, for example: Logistic Regression]
* [Model 2, for example: Random Forest]

After training, I compare the results to see which model catches the most fraud without making too many mistakes on normal transactions.
